## What's New in v1.1.0

### Localization Support
- Support for 6 languages: Russian, English, Ukrainian, German, French, Spanish
- Automatic language detection based on current Windows keyboard layout
- Language selection in application settings

### Smart Language Detection
Status messages (Ready, Listening, Recognizing, Done, etc.) now automatically display in the language of the current keyboard layout.

### Documentation
- Added i18n module for translation management
- Added localization documentation (LOCALIZATION.md)
